# Water Consumption Forecasting — PERUMDA Air Minum Tirta Tamiang

Forecasting clean water consumption for customers of PERUMDA Air Minum Tirta Tamiang
using the ARIMA (Autoregressive Integrated Moving Average) method,
implemented in Python (Google Colab). 

## Customer Categories
- **Golongan I** — Social facilities (hydrants, places of worship, orphanages)
- **Golongan II** — Residential (Rumah Tangga A, B, C, D)
- **Golongan III** — Government & commercial institutions

## Tech Stack
- **Language:** Python (Google Colab)
- **Method:** ARIMA (statsmodels)
- **Libraries:** pandas, numpy, matplotlib, statsmodels

## Notebooks
| File | Category |
| arima-golongan-1-fasilitas-sosial.ipynb | Social facilities |
| arima-golongan-2-rumah-tangga.ipynb | Residential |
| arima-golongan-3-instansi-niaga.ipynb | Government & commercial |

## Background
Undergraduate thesis project at Universitas Malikussaleh (2026), 
aimed at helping PDAM Tirta Tamiang optimize water distribution 
planning through data-driven forecasting.
